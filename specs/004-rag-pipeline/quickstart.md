# Quickstart Guide: RAG Pipeline\n\n## Setup\n\n1. Ensure you have the following environment variables set:\n   ```bash\n   QDRANT_HOST=your-qdrant-host\n   QDRANT_API_KEY=your-qdrant-api-key\n   OPENAI_API_KEY=your-openai-api-key\n   ```\n\n2. Install required dependencies:\n   ```bash\n   pip install -r requirements.txt\n   ```\n\n## Running the RAG Pipeline\n\n### 1. Using the RAG Pipeline Class\n\n```python\nfrom rag_pipeline import RAGPipeline\n\n# Initialize the pipeline\nrag = RAGPipeline(top_k=5, context_window_size=3000)\n\n# Query the system\nresult = rag.query(\"What is ROS 2 and how does it work?\")\n\nprint(\"Answer:\", result['answer'])\nprint(\"Sources:\", result['sources'])\n```\n\n### 2. Using the API\n\nSend a POST request to `/ask`:\n\n```bash\ncurl -X POST http://localhost:8000/ask \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n    \"query\": \"What is ROS 2 and how does it work?\"\n  }'\n```\n\n## Key Components\n\n1. **Semantic Search**: Performs vector similarity search in Qdrant\n2. **Context Packing**: Aggregates relevant documents while managing context limits\n3. **Response Generation**: Uses language model to generate answers based on context\n
